# Infinite Forward Ball
### 무한으로 앞으로 가는 공


**작동방법**
1) 마우스를 클릭하면 왼쪽에서 오른쪽으로 부는 바람이 적용된다.
2) 오른쪽 벽은 막혀있지 않아 통과할 수 있다.
3) 통과하면 왼쪽 벽에서 나와서 무한으로 앞으로 간다.
